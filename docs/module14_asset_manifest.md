## Download Instructions

All assets are available via Figma MCP asset URLs (valid for 7 days). To download:

1. Create directory: `mkdir -p assets/blank_states/icons`
2. Download each asset using curl or wget
3. Ensure all assets are added to `pubspec.yaml` under `assets:`

## pubspec.yaml Entry

```yaml
flutter:
  assets:
    - assets/blank_states/
```
