1. Open each URL above in your browser
2. Save image as PNG to the corresponding `save_as` path
3. Ensure folder structure exists: `assets/rent_vehicle/categories/`, `assets/rent_vehicle/banners/`, `assets/rent_vehicle/vehicles/`

### Option 2: Automated Download (PowerShell Script)
Use the provided script in `scripts/download_module4_assets.ps1`:

